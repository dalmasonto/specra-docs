## Probieren Sie es selbst

1. **Erweitern Sie einen beliebigen Endpunkt**, indem Sie darauf klicken
2. **Scrollen Sie zum Playground** am Ende jedes Endpunkts
3. **Klicken Sie auf "Request senden"**, um die API live zu testen
4. **Sehen Sie sich die Antwort** im JSON-Viewer an

## Über JSONPlaceholder

[JSONPlaceholder](https://jsonplaceholder.typicode.com/) ist eine kostenlose gefälschte REST-API zum Testen und Prototyping. Es ist perfekt für:

- Testen von API-Dokumentationstools (wie diesem!)
- Frontend-Entwicklung
- Lernen von HTTP-Methoden
- Prototyping von Anwendungen

<Callout type="success" title="Profi-Tipp">
Versuchen Sie, den Request-Body in den POST/PUT/PATCH-Endpunkten zu ändern, und sehen Sie, wie sich die Antwort ändert!
</Callout>

## Beispiel-Integration

So könnten Sie diese API in einer echten Anwendung verwenden:

```javascript
// Alle Beiträge abrufen
async function getPosts() {
  const response = await fetch('https://jsonplaceholder.typicode.com/posts?_limit=5');
  const posts = await response.json();
  return posts;
}

// Einen neuen Beitrag erstellen
async function createPost(title, body) {
  const response = await fetch('https://jsonplaceholder.typicode.com/posts', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      title,
      body,
      userId: 1
    })
  });
  return await response.json();
}

// Verwendung
const posts = await getPosts();
console.log(posts);

const newPost = await createPost('My Title', 'My Content');
console.log(newPost);
```
