## Sobre JSONPlaceholder

[JSONPlaceholder](https://jsonplaceholder.typicode.com/) es una API REST falsa gratuita para pruebas y creación de prototipos. Es perfecto para:

- Probar herramientas de documentación de API (¡como esta!)
- Desarrollo frontend
- Aprender métodos HTTP
- Prototipar aplicaciones

<Callout type="success" title="Consejo Pro">
¡Intente modificar el cuerpo de la solicitud en los puntos finales POST/PUT/PATCH y vea cambiar la respuesta!
</Callout>

## Ejemplo de integración

Así es como podría usar esta API en una aplicación real:

```javascript
// Obtener todas las publicaciones
async function getPosts() {
  const response = await fetch('https://jsonplaceholder.typicode.com/posts?_limit=5');
  const posts = await response.json();
  return posts;
}

// Crear una nueva publicación
async function createPost(title, body) {
  const response = await fetch('https://jsonplaceholder.typicode.com/posts', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      title,
      body,
      userId: 1
    })
  });
  return await response.json();
}

// Uso
const posts = await getPosts();
console.log(posts);

const newPost = await createPost('My Title', 'My Content');
console.log(newPost);
```
