## À propos de JSONPlaceholder

[JSONPlaceholder](https://jsonplaceholder.typicode.com/) est une fausse API REST gratuite pour les tests et le prototypage. C'est parfait pour :

- Tester des outils de documentation API (comme celui-ci !)
- Développement frontend
- Apprendre les méthodes HTTP
- Prototyper des applications

<Callout type="success" title="Conseil de pro">
Essayez de modifier le corps de la requête dans les points de terminaison POST/PUT/PATCH et voyez la réponse changer !
</Callout>

## Exemple d'intégration

Voici comment vous pourriez utiliser cette API dans une application réelle :

```javascript
// Récupérer tous les articles
async function getPosts() {
  const response = await fetch('https://jsonplaceholder.typicode.com/posts?_limit=5');
  const posts = await response.json();
  return posts;
}

// Créer un nouvel article
async function createPost(title, body) {
  const response = await fetch('https://jsonplaceholder.typicode.com/posts', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      title,
      body,
      userId: 1
    })
  });
  return await response.json();
}

// Utilisation
const posts = await getPosts();
console.log(posts);

const newPost = await createPost('My Title', 'My Content');
console.log(newPost);
```
