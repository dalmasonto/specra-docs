El componente Mermaid renderiza diagramas a partir de texto utilizando la biblioteca Mermaid.js.

## Uso

```mdx
<Mermaid
  chart={`
    graph TD
      A[Inicio] --> B{Decisión}
      B -->|Sí| C[Acción 1]
      B -->|No| D[Acción 2]
  `}
  caption="Diagrama de flujo de decisión"
/>
```

## Props

| Propiedad | Tipo | Predeterminado | Descripción |
|-----------|------|----------------|-------------|
| `chart` | `string` | requerido | Definición del diagrama Mermaid |
| `caption` | `string` | - | Leyenda debajo del diagrama |

## Tipos de diagramas

Mermaid admite muchos tipos de diagramas:

- **Diagramas de flujo** - `graph TD` o `graph LR`
- **Diagramas de secuencia** - `sequenceDiagram`
- **Diagramas de clases** - `classDiagram`
- **Diagramas de estado** - `stateDiagram-v2`
- **Entidad Relación** - `erDiagram`
- **Diagramas de Gantt** - `gantt`
- **Gráficos circulares** - `pie`

## Soporte de temas

<Callout type="info">
Los diagramas Mermaid se adaptan automáticamente a los temas claro y oscuro.
</Callout>
