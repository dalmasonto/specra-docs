Las tarjetas de imagen combinan imágenes con superposiciones de texto, perfectas para galerías, exhibiciones de características o navegación visual.

## Uso

```mdx
<ImageCard
  src="/images/feature.png"
  alt="Captura de pantalla de característica"
  title="Característica increíble"
  description="Descripción de la característica"
  href="/docs/v1.0.0/features"
/>
```

## Props

### ImageCard

| Propiedad | Tipo | Predeterminado | Descripción |
|-----------|------|----------------|-------------|
| `src` | `string` | requerido | URL de origen de imagen |
| `alt` | `string` | requerido | Texto alternativo accesible |
| `title` | `string` | - | Título de la tarjeta |
| `description` | `string` | - | Descripción de la tarjeta |
| `href` | `string` | - | URL del enlace (hace que la tarjeta sea clicable) |
| `external` | `boolean` | `false` | Abre el enlace en una nueva pestaña |
| `aspectRatio` | `"square" \| "video" \| "portrait"` | `"video"` | Relación de aspecto de imagen |

### ImageCardGrid

| Propiedad | Tipo | Predeterminado | Descripción |
|-----------|------|----------------|-------------|
| `children` | `ReactNode` | requerido | Componentes ImageCard |
| `cols` | `1 \| 2 \| 3 \| 4` | `3` | Número de columnas |
