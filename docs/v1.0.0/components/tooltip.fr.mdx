Les infobulles affichent des informations supplémentaires lors du survol d'un élément.

## Utilisation

```mdx
Survolez <Tooltip content="Ceci est une infobulle !">ce texte</Tooltip> pour en voir plus.
```

## Exemples en direct

### Infobulle basique

Survolez <Tooltip content="Le contexte supplémentaire apparaît ici">ce texte</Tooltip> pour voir une infobulle.

## Props

| Prop | Type | Défaut | Description |
|------|------|--------|-------------|
| `children` | `ReactNode` | requis | Élément qui déclenche l'infobulle |
| `content` | `string` | requis | Contenu texte de l'infobulle |
| `position` | `"top" \| "bottom" \| "left" \| "right"` | `"top"` | Position de l'infobulle |

## Meilleures pratiques

<Callout type="tip">
- Gardez le contenu de l'infobulle concis (1-2 phrases max)
- Utilisez pour des informations supplémentaires, pas pour du contenu critique
- Assurez-vous que l'élément déclencheur est accessible au clavier
</Callout>

## Modèles courants

### Icônes d'aide

Ajoutez de petites icônes d'aide à côté des étiquettes de formulaire ou des fonctionnalités :

```mdx
**Email** <Tooltip content="Nous ne partagerons jamais votre email">
  <Icon icon="help-circle" size={14} />
</Tooltip>
```

### Abréviations

Expliquez les abréviations en ligne :

```mdx
L'<Tooltip content="Interface de Programmation d'Application">API</Tooltip> est bien documentée.
```
