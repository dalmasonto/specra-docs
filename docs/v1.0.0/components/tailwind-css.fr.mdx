Le composant Tailwind CSS vous permet d'appliquer facilement des classes utilitaires Tailwind CSS pour styliser le contenu de votre documentation. Utilisez simplement des éléments **HTML** standard avec des **classes Tailwind**.

## Utilisation

<Tabs defaultValue="Code">
  <Tab label="Code">
```mdx
<div className="bg-gray-200/15 p-4 rounded">
  Ce contenu est stylisé avec Tailwind CSS !
</div>
```
    </Tab>
    
    <Tab label="Aperçu">
        <div className="bg-gray-200/15 p-4 rounded">
        Ce contenu est stylisé avec Tailwind CSS !
        </div>
    </Tab>
</Tabs>
