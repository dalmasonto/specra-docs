The Tailwind CSS component allows you to easily apply Tailwind CSS utility classes to style your documentation content. Just use standard **HTML** elements with **Tailwind classes**.

## Usage

<Tabs defaultValue="Code">
  <Tab label="Code">
```mdx
<div className="bg-gray-200/15 p-4 rounded">
  This content is styled with Tailwind CSS!
</div>
```
    </Tab>
    
    <Tab label="Preview">
        <div className="bg-gray-200/15 p-4 rounded">
        This content is styled with Tailwind CSS!
        </div>
    </Tab>
</Tabs>
