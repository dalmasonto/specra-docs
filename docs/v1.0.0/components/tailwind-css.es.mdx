El componente Tailwind CSS le permite aplicar fácilmente clases de utilidad de Tailwind CSS para estilizar el contenido de su documentación. Simplemente use elementos **HTML** estándar con **clases de Tailwind**.

## Uso

<Tabs defaultValue="Code">
  <Tab label="Código">
```mdx
<div className="bg-gray-200/15 p-4 rounded">
  ¡Este contenido está estilizado con Tailwind CSS!
</div>
```
    </Tab>
    
    <Tab label="Vista previa">
        <div className="bg-gray-200/15 p-4 rounded">

        </div>
    </Tab>
</Tabs>
