Die Icon-Komponente zeigt Icons aus verschiedenen Quellen an, einschließlich Lucide-Icons, Font Awesome und benutzerdefinierten Bildern.

## Verwendung

```mdx
<Icon icon="rocket" size={24} color="#3b82f6" />
```

## Props

| Prop | Typ | Standard | Beschreibung |
|------|-----|----------|--------------|
| `icon` | `string \| ReactNode` | erforderlich | Icon-Name, URL oder React-Node |
| `iconType` | `string` | `"regular"` | Font Awesome Stil (falls FA verwendet wird) |
| `color` | `string` | - | Icon-Farbe (CSS-Farbwert) |
| `size` | `number` | `20` | Icon-Größe in Pixeln |
| `className` | `string` | - | Zusätzliche CSS-Klassen |

## Icon-Quellen

### Lucide Icons (Standard)

Verwenden Sie Kebab-Case-Icon-Namen von [Lucide](https://lucide.dev/icons):

```mdx
<Icon icon="arrow-right" />
<Icon icon="check-circle" />
<Icon icon="external-link" />
```

### Bild-URLs

Verwenden Sie eine beliebige Bild-URL oder lokalen Pfad:

```mdx
<Icon icon="/images/logo.svg" size={24} />
<Icon icon="https://example.com/icon.png" size={24} />
```

### Font Awesome

Präfixen Sie Icons mit `fa-`, um Font Awesome zu verwenden:

```mdx
<Icon icon="fa-github" iconType="brands" />
```
