Les cartes image combinent des images avec des superpositions de texte, parfaites pour les galeries, les présentations de fonctionnalités ou la navigation visuelle.

## Utilisation

```mdx
<ImageCard
  src="/images/feature.png"
  alt="Capture d'écran de fonctionnalité"
  title="Fonctionnalité incroyable"
  description="Description de la fonctionnalité"
  href="/docs/v1.0.0/features"
/>
```

## Props

### ImageCard

| Prop | Type | Défaut | Description |
|------|------|--------|-------------|
| `src` | `string` | requis | URL source de l'image |
| `alt` | `string` | requis | Texte alternatif accessible |
| `title` | `string` | - | Titre de la carte |
| `description` | `string` | - | Description de la carte |
| `href` | `string` | - | URL du lien (rend la carte cliquable) |
| `external` | `boolean` | `false` | Ouvre le lien dans un nouvel onglet |
| `aspectRatio` | `"square" \| "video" \| "portrait"` | `"video"` | Ratio d'aspect de l'image |

### ImageCardGrid

| Prop | Type | Défaut | Description |
|------|------|--------|-------------|
| `children` | `ReactNode` | requis | Composants ImageCard |
| `cols` | `1 \| 2 \| 3 \| 4` | `3` | Nombre de colonnes |
