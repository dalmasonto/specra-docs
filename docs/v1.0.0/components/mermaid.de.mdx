Die Mermaid-Komponente rendert Diagramme aus Text mithilfe der Mermaid.js-Bibliothek.

## Verwendung

```mdx
<Mermaid
  chart={`
    graph TD
      A[Start] --> B{Entscheidung}
      B -->|Ja| C[Aktion 1]
      B -->|Nein| D[Aktion 2]
  `}
  caption="Entscheidungsflussdiagramm"
/>
```

## Props

| Prop | Typ | Standard | Beschreibung |
|------|-----|----------|--------------|
| `chart` | `string` | erforderlich | Mermaid-Diagramm-Definition |
| `caption` | `string` | - | Bildunterschrift unter dem Diagramm |

## Diagrammtypen

Mermaid unterstützt viele Diagrammtypen:

- **Flussdiagramme** - `graph TD` oder `graph LR`
- **Sequenzdiagramme** - `sequenceDiagram`
- **Klassendiagramme** - `classDiagram`
- **Zustandsdiagramme** - `stateDiagram-v2`
- **Entity Relationship** - `erDiagram`
- **Gantt-Diagramme** - `gantt`
- **Tortendiagramme** - `pie`

## Theme-Support

<Callout type="info">
Mermaid-Diagramme passen sich automatisch an helle und dunkle Themes an.
</Callout>
