Cards create visually distinct content blocks, perfect for feature lists, navigation links, or highlighted content.

## Usage

```mdx
<Card 
  title="Getting Started" 
  description="Learn the basics of our platform"
  href="/docs/v1.0.0/getting-started"
/>
```

## Live Examples

### Basic Card

<Card 
  title="Documentation" 
  description="Read the full documentation for detailed guides and API references."
/>

### Linked Card

<Card

<Card

### External Link

<Card 
  title="GitHub Repository" 
  description="View the source code and contribute"
  href="https://github.com"

/>

## Card Grid

Use `CardGrid` to display multiple cards in a responsive grid layout:

```mdx
<CardGrid cols={2}>
  <Card title="Card 1" description="First card" />
  <Card title="Card 2" description="Second card" />
</CardGrid>
```

<CardGrid cols={2}>
  <Card icon="rocket" title="Quick Start" description="Get up and running in 5 minutes" href="/docs/v1.0.0/getting-started/quickstart" />
  <Card icon="book" title="Guides" description="In-depth tutorials and walkthroughs" href="/docs/v1.0.0/guides" />
  <Card icon="code" title="API Reference" description="Complete API documentation" />
  <Card icon="github" title="GitHub" description="Contribute to the project" href="https://github.com" external />
</CardGrid>

## Props

### Card

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | required | Card title |
| `description` | `string` | - | Card description text |
| `href` | `string` | - | Link URL (makes card clickable) |
| `icon` | `string \| ReactNode` | - | Lucide icon name or custom icon |
| `external` | `boolean` | `false` | Opens link in new tab |
| `children` | `ReactNode` | - | Custom content |

### CardGrid

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | required | Card components |
| `cols` | `1 \| 2 \| 3` | `2` | Number of columns |
