Bildkarten kombinieren Bilder mit Textüberlagerungen, perfekt für Galerien, Funktionspräsentationen oder visuelle Navigation.

## Verwendung

```mdx
<ImageCard
  src="/images/feature.png"
  alt="Feature Screenshot"
  title="Erstaunliches Feature"
  description="Beschreibung des Features"
  href="/docs/v1.0.0/features"
/>
```

## Props

### ImageCard

| Prop | Typ | Standard | Beschreibung |
|------|-----|----------|--------------|
| `src` | `string` | erforderlich | Bildquellen-URL |
| `alt` | `string` | erforderlich | Barrierefreier Alt-Text |
| `title` | `string` | - | Kartentitel |
| `description` | `string` | - | Kartenbeschreibung |
| `href` | `string` | - | Link-URL (macht Karte anklickbar) |
| `external` | `boolean` | `false` | Öffnet Link in neuem Tab |
| `aspectRatio` | `"square" \| "video" \| "portrait"` | `"video"` | Bildseitenverhältnis |

### ImageCardGrid

| Prop | Typ | Standard | Beschreibung |
|------|-----|----------|--------------|
| `children` | `ReactNode` | erforderlich | ImageCard-Komponenten |
| `cols` | `1 \| 2 \| 3 \| 4` | `3` | Anzahl der Spalten |
