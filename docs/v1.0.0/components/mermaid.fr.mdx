Le composant Mermaid rend des diagrammes à partir de texte en utilisant la bibliothèque Mermaid.js.

## Utilisation

```mdx
<Mermaid
  chart={`
    graph TD
      A[Début] --> B{Décision}
      B -->|Oui| C[Action 1]
      B -->|Non| D[Action 2]
  `}
  caption="Organigramme de décision"
/>
```

## Props

| Prop | Type | Défaut | Description |
|------|------|--------|-------------|
| `chart` | `string` | requis | Définition du diagramme Mermaid |
| `caption` | `string` | - | Légende sous le diagramme |

## Types de diagrammes

Mermaid prend en charge de nombreux types de diagrammes :

- **Organigrammes** - `graph TD` ou `graph LR`
- **Diagrammes de séquence** - `sequenceDiagram`
- **Diagrammes de classe** - `classDiagram`
- **Diagrammes d'état** - `stateDiagram-v2`
- **Entité Relation** - `erDiagram`
- **Diagrammes de Gantt** - `gantt`
- **Diagrammes circulaires** - `pie`

## Support des thèmes

<Callout type="info">
Les diagrammes Mermaid s'adaptent automatiquement aux thèmes clair et sombre.
</Callout>
