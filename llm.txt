# Specra Docs - AI-Friendly Documentation

## Project Overview
specra-docs is the official documentation website for Specra. It serves as both comprehensive documentation for users and a reference implementation demonstrating Specra's capabilities.

## Key Information
- Project Name: specra-docs
- Type: Documentation Website / Next.js Application
- Purpose: Document Specra framework and demonstrate its features
- URL: https://specra.vercel.app
- Repository: https://github.com/dalmasonto/specra-docs
- Built With: Specra (dogfooding)

## Core Purpose
1. **Documentation**: Comprehensive guides, API references, and tutorials for Specra
2. **Reference Implementation**: Real-world example of Specra in production
3. **Showcase**: Demonstrates all Specra features and capabilities
4. **Onboarding**: Helps new users learn and adopt Specra

## Technology Stack
- Next.js 16.1.0 (App Router)
- React 19.2.3
- Specra 0.1.7 (the SDK this project documents)
- Tailwind CSS 4.1.9
- TypeScript 5
- Deployed on Vercel

## Build Scripts
- `dev`: Development server (next dev)
- `build`: Production build with default config
- `build:export`: Static export build for GitHub Pages
- `start`: Production server
- `lint`: ESLint
- `generate:redirects`: Generate redirect rules
- `generate:static-redirects`: Generate static redirects for export
- `index:search`: Index content in MeiliSearch
- `test:search`: Test search functionality

## Deployment Options
1. **Vercel (Default)**: `npm run build` - Server-side rendering
2. **GitHub Pages**: `npm run build:export` - Static site export
3. **Netlify**: Compatible with standard build
4. **Self-hosted**: Any Node.js hosting

## Static Export Configuration
For GitHub Pages deployment:
- Uses `NEXT_PUBLIC_BASE_PATH` environment variable
- Generates static HTML files
- Creates redirect files for routing
- Supports custom domain or subdirectory hosting

## Relationship to Other Projects
- **specra-sdk**: This project uses specra as a dependency to build itself
- **specra-cli**: Documented here as a tool users can use
- **Flow**: SDK provides functionality → Docs explain how to use it → CLI helps users start

**Circular Relationship**:
- Specra SDK enables documentation sites
- This site documents the SDK
- The site is built with the SDK it documents (dogfooding)

## Search Integration
MeiliSearch configuration (currently disabled):
- Host: http://localhost:7700
- API Key: aSampleMasterKey
- Index Name: docs

To enable:
1. Set `search.enabled: true` in specra.config.json
2. Run MeiliSearch instance
3. Run `npm run index:search` to index content
4. Search UI automatically appears
